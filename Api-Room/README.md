## Rotas criados na API

* Criar sala de reuniao
```shell script
POST - /api/v1/rooms
```

* Listar todas as salas
```shell script
GET - /api/v1/rooms
```

* Buscar uma sala pelo Id
```shell script
GET - /api/v1/rooms/{id}
```

* Atualizar uma sala pelo Id
```shell script
PUT - /api/v1/rooms/{id}
```

* Excluir uma sala pelo id
```shell script
DELETE - /api/v1/rooms/{Id}
```
